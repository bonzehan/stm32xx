STM32++
=======

C++ utility library for STM32 software developers.

Testing
-------

Generating Unit Test runner
^^^^^^^^^^^^^^^^^^^^^^^^^^^

.. code-block::

    scons unit-test

Running unit tests
^^^^^^^^^^^^^^^^^^

.. code-block::

    ./build/test/unit/run-tests

Currently the unit tests are compiled and run on host only (not on target).
